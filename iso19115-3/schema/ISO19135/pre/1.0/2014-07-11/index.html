Procedures for Registration (PRE) Version: 1.0

Description

PRE 1.0 is an XML Schema implementation derived from ISO ISO 19135, Geographic information -- Procedures for item registration. It includes abstract classes shared by multiple namespaces and is required in order to enable modularization of the XML implementation. The XML schema was encoded using the rules described in ISO/TS 19139:2007, Clause 8 and ISO/TS19115-3, Clause 8.

XML Namespace for pre 1.0

The namespace URI for pre 1.0 is http://www.isotc211.org/namespace/pre/1.0.

XML Schema for pre 1.0

pre.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the pre 1.0 namespace or by XML Schema documents importing the pre 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the pre namespace, but it does not contain the declaration of any types.

NOTE: The XML Schema for pre 1.0 are available here as part of a zip archive including all the XML Schema Implementations defined in ISO/TS 19115-3.

Related XML Schema for pre 1.0

abstract.xsd implements the UML conceptual schema defined in ISO 19135, Geographic information -- Procedures for item registration. It was created using the encoding rules defined in ISO 19118, ISO 19139, and ISO 19115-3 and contains the following classes: Abstract_Register, and Abstract_RegisterItem
